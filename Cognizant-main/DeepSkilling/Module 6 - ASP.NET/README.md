# Module 6 - ASP.NET Core Web API

ASP.NET Core Web API hands-on exercises.

## Exercises

1. `Exercise-01-FirstWebApi` - basic read/write Web API actions.
2. `Exercise-02-SwaggerEmployeeApi` - Swagger-enabled Employee API with `Emp` route.
3. `Exercise-03-ModelsAndFilters` - custom Employee model, auth filter, exception filter.
4. `Exercise-04-EmployeeCrud` - Employee update through HTTP PUT with validation.
5. `Exercise-05-JwtCorsSecurity` - CORS, JWT authentication, and role authorization.
6. `Exercise-06-KafkaChat` - Kafka chat producer and consumer console apps.

Run an API exercise:

```powershell
dotnet run --project .\Exercise-01-FirstWebApi\Exercise-01-FirstWebApi.csproj
```

Swagger is available at `/swagger`.
